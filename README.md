# miR-TS Manuscript

This repository contains processed data and analysis code to reproduce
all results and figures in the manuscript:

"Circulating extracellular microRNAs as tissue-specific biomarkers of human health and disease"

## Contents
- Processed miRNA expression data
- Computed miR-TS scores
- Analysis code for all figures

## Data sources
Raw sequencing data are available from NCBI SRA:
- PRJNA1259993
- PRJNA1258725
- PRJNAxxxxxxx

## Reproducibility
To reproduce all figures:

1. Install the miRTS R package
2. Run scripts in `code/` in numerical order
3. Figures will be generated in `figures/generated_figures/`

## Software environment
R version and package versions are provided in `environment/sessionInfo.txt`.
